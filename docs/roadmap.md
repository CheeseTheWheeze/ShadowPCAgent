# Roadmap

## Phase 1: Foundations

- Orchestrator + Planner scaffolding.
- Executor interfaces for code, CLI, and GUI.
- Initial logging and audit primitives.

## Phase 2: Code intelligence

- Repo scanning and indexing.
- Patch/diff editing and formatting hooks.
- Test discovery and execution strategy.

## Phase 3: GUI automation

- Screen capture and OCR.
- UI element detection and scene graph.
- Action primitives with validation.

## Phase 4: Safety & approvals

- Sensitive change detection.
- Draft/diff workflow.
- Command allowlist enforcement.

## Phase 5: Reliability

- Retry logic and fallback selectors.
- Performance tuning for reaction time.

## Phase 6: Release readiness

- Integration tests and performance benchmarks.
- Documentation and user guides.
