# Operations

## Logging

- All actions (CLI + GUI) are logged.
- Store timestamps, inputs, and outputs.
- Maintain a deterministic record for audit.

## Auditing

- Store diffs and artifacts for each task.
- Provide run summaries and rollback guidance.

## Configuration

- Configurable command allowlists.
- Adjustable GUI reaction intervals.
- Memory/retention policies for context data.
